Hadoop 2 与 3 在配置环境时的区别

· 2024-03-21 · # hadoop

区别不是很多（在环境配置方面）

一、hdfs 网页面板

从 hadoop 3 开始 master:50070 更为 master:9870

master 为主机的主机名

一、slaves文件

从 hadoop 3 开始 slaves 文件更名为 workers

为反对种族歧视，几乎各大知名项目都有类似行为
比如 Github 分支名 master 改为 main

其他方面目前没有发现
