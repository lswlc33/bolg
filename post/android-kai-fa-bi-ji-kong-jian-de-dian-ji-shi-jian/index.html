Android开发笔记——控件的点击事件

· 2023-04-08 ·

给按钮添加一个点击事件
经过测试，不单单是按钮，图片等其他控件也可以设置

创建对象

Button button1 = (Button) findViewById(R.id.button);
// 创建Button对象
## 单击事件

button1.setOnClickListener(new View.OnClickListener() {
    // 创建点击监听器
    @Override
    public void onClick(View view) {
        // 要执行的点击事件
    }
});

双击事件

button1.setOnDoubleClickListener(new View.OnDoubleClickListener() {
    @Override
    public void onDoubleClick(View v) {
        // 双击事件操作
    }
});
